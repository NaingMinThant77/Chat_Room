# Chat_Room
- This is real time socket-io app. 
- Frontend - React, Tailwind, socket.io-client. 
- Backend - ExpressJs, socket.io. Database - Mongodb


## Deployment

To use this project run on browser

```bash
  https://chat-room-6ytc.onrender.com
```

or

To run in app
```bash
  npm run dev
```
